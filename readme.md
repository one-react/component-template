## Usage

Install [SAOJS](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
cd my-component
sao or-template
```

### From git

```bash
cd my-component
sao one-react/component-template
```
